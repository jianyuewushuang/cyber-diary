# **【生活娱乐赛道】赛博日记本**

## **1. 创意名称 + 创意介绍**

- 创意的名称叫赛博日记本
- 我在平时平时写日记时发现写了很多日记之后很难直观感受写好的日记，每次点开查看太麻烦，于是便想做一个可视化赛博日记本，方便查看日记和统计日记数据
- 产品会是一个命令行工具，通过几条简单的命令把仓库中的日记做成精美的html预览、查看和统计页面

## **2. 目标用户及痛点**

- 面向经常写电子日记，并追求对文件的掌握权的用户
- 当用户不想用复杂的笔记软件进行日记管理，追求完全自主掌握日记文件，而又苦于难以对大量日记文件进行整理统计和快速查看时，这个产品会发挥很大作用
- 如果没有我的产品，用户对本地日记文件的管理非常不方便，难以做到直观可视化感知，想查看和搜索日记也非常困难

## **3. 价值与意义**

- 这款产品可以帮助用户管理日记，让日记对用户生活的记录发挥更大意义

## **4. 产品功能和实现**

- 把/diary文件夹存放日记文件
- 只识别/diary文件夹下的markdown格式的日记文件
- 日记文件的文件名必须为日期，格式为yyyyMMdd.md，例如20250801.md，20260122.md等
- 把/diary中所有markdown格式的日记文件转换为html、css和js文件，放到/build文件夹下
- html打开之后会以日历的形式展示日记，对应日期有日记文件就用圆圈对对应日期进行标记，点击日期可以查看对应文件的全部内容
- 日记文件内容越多，日记上对应日期的圆圈越大
- 生成的html文件可以对日记的开头进行预览，点击后可以查看日记的全部内容
- 预览时会显示日记的字数
- 可以在浏览器中根据日记内容搜索日记
- 统计日记数据，包括每周、每月、每年日记总数，每周、每月、每年日记字数，以及每个日期的字数，以可视化图表的形式展示
- 可以在浏览器中查看统计图表，点击图表可以查看对应日期的全部内容
- 页面风格统一为半透明方角，主题色为紫色

## **5. 产品使用**

```bash
# 构建生成 HTML
npm run build
```
